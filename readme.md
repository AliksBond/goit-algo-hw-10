Висновки щодо правильності розрахунків:

Метод Монте-Карло для обчислення визначеного інтеграла показав результат, який узгоджується з аналітичним розрахунком та результатом функції quad з бібліотеки SciPy.

Невелика похибка обумовлена випадковою природою методу та обмеженою кількістю випадкових точок.

Порівняння результатів підтвердило, що реалізований алгоритм працює коректно та забезпечує точне наближення інтеграла.

Це демонструє ефективність методу Монте-Карло для чисельного обчислення інтегралів, особливо у випадках, коли аналітичне інтегрування ускладнене або неможливе.
